# Tailwind CSS Project Schema

You can start Project just clone this repository

## Installation

Install tailwind-html with npm

```bash
   git clone https://github.com/project-schema/tailwind-html.git
```

```bash
   cd tailwind-html
```

then,

```bash
   npm i
```

## Deployment

Run this porject

```bash
   npm run w
```

Build this porject for optimize css

```bash
   npm run build
```
